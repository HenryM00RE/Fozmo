# Fozmo 0.0.1 — public pre-alpha

This is the first public pre-alpha snapshot of Fozmo, a local-first music
player and DSP host for macOS.

## Included in this build

- Local-library and Qobuz browsing and playback.
- CoreAudio PCM output, sample-rate conversion, parametric EQ, and
  hardware-dependent PCM-to-DSD output.
- Browser control from the Mac and other devices on an explicitly enabled
  local network, including an iPhone-tested control path.
- Experimental Sonos, AirPlay, Hegel, UPnP, and remote-agent integrations for
  wider hardware testing.
- Queue, playlist, settings, and library persistence with recovery paths.

## Pre-alpha boundaries

- Hardware and receiver compatibility is intentionally narrow and is not
  guaranteed beyond tested setups. PCM is the safest output path.
- DSD rates and transport modes depend on the DAC, operating system, and
  driver. DSD256/DoP remains experimental.
- AirPlay 2 groups, stereo pairs, password/PIN-protected receivers, and
  FairPlay-only receivers are not supported.
- Apple Music capture, a native iOS app, native SACD support, Windows release
  artifacts, and offline mobile-library playback are not part of 0.0.1.
- This DMG is ad-hoc signed, not Apple Developer ID signed or notarized.
  Automatic update checks are disabled.

Please treat this as test software: keep backups of the library and settings,
and report the macOS version, hardware/output path, and logs with any issue.
